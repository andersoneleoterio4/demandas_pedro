# Controle de Demandas - Pedro

Página web simples para visualização de demandas e processos da ANM.

## Sobre

A aplicação (`index.html`) busca os dados de uma API publicada no Google Apps Script e os exibe em uma tabela com as seguintes colunas:

- Cliente
- CNPJ/CPF
- Processo ANM
- NUP
- Demanda
- Status

## Como usar

Basta abrir o arquivo `index.html` em um navegador. Os dados são carregados automaticamente a partir da API configurada na constante `API` dentro do arquivo.
